お手元のスマホからお子さまの小テスト結果を入力いただくと、自動で弱点分析と復習教材が作成されます。

## 3. 保守管理者向け投稿

「まなびチェック」のプロトタイプを展開しました。

**技術スタック**
- Next.js 16 (App Router / TypeScript / Tailwind CSS)
- Supabase (PostgreSQL + Storage)
- shadcn/ui

**リポジトリ**
→ https://github.com/camelllllllllll/manabi-check

**本番URL**
→ https://manabi-check.vercel.app

**ドキュメント**
- セットアップ手順・テーブル定義・環境変数：添付の保守マニュアルを参照
- SQLマイグレーション: `supabase/migrations/001_create_tables.sql`
- サンプルデータ: `supabase/seed.sql`

**現状の制約（MVP段階）**
- 認証はアクセスコードのみ（Cookieベースの簡易認証）
- RLSは全テーブルALL許可（本番前に要修正）
- 画像ストレージはpublicバケット
- PDF出力はブラウザ印刷機能を利用

**保守管理マニュアル**
