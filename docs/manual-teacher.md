# まなびチェック 講師向け操作マニュアル

## はじめに

「まなびチェック」は、小テストの採点・弱点分析・復習教材生成を自動化するシステムです。
講師は管理画面から問題の登録、テストの作成、生徒の管理を行います。

> 現在はプロトタイプ段階です。操作中に不具合があればフィードバックをお願いします。

---

## 管理画面へのアクセス

ブラウザで以下を開いてください：

https://manabi-check.vercel.app/admin/units

上部のタブで各管理画面を切り替えられます。

---

## 1. 単元管理

**場所：** 管理画面 → 「単元管理」タブ

### 教科を追加する
1. 右上の「教科を追加」をクリック
2. 教科名（例：社会）を入力して「追加」

### 単元を追加する
1. 上部の教科プルダウンから教科を選択
2. 「単元を追加」をクリック
3. 単元名（例：日本の地形）を入力
4. 必要に応じて親単元を選択（大単元→小単元の階層構造が作れます）
5. 表示順を設定して「追加」

### 単元を編集・削除する
- 各単元の右側にある「編集」「削除」ボタンから操作

---

## 2. 問題マスタ管理

**場所：** 管理画面 → 「問題管理」タブ

### 問題を登録する
1. 右上の「問題を追加」をクリック
2. 以下の項目を入力：
   - **教科**（必須）：プルダウンから選択
   - **単元**（必須）：プルダウンから選択
   - **問題形式**（必須）：選択式 / 穴埋め / 記述式 / その他
   - **問題文**（任意）：管理用。保護者画面には表示されません
   - **正答**（必須）：正解の文字列
   - **図表画像**（任意）：画像ファイルをアップロード
   - **解説**（任意）：復習教材に表示されます
   - **復習ポイント**（任意）：1行に1つずつ入力
   - **難易度**（任意）：1〜5
3. 「登録」をクリック

### 問題を検索・フィルタする
- 画面上部のフィルタで教科・単元・問題形式・キーワードで絞り込み可能
- 各問題にはDB通し番号（No.）が自動で振られます

### 問題を編集・削除する
- 各問題の右側にある「編集」「削除」ボタンから操作

---

## 3. テスト管理

**場所：** 管理画面 → 「テスト管理」タブ

### テストを作成する
1. 右上の「テストを作成」をクリック
2. 基本情報を入力：
   - **テスト名**（例：第5回 社会小テスト）
   - **教科**
   - **実施日**
3. 「保護者に公開する」を必要に応じてONに
4. 問題構成を設定：
   - 「問題を追加」で問題マスタから選択
   - 各問題の **配点** を設定（デフォルト1点）
   - ↑↓ボタンで並び替え、×ボタンで削除
   - **合計配点が15点になるように調整**（15点でないと保存できません）
5. 「テストを作成」をクリック

### テストの公開/非公開
- テスト一覧の「公開」「非公開」ボタンで切り替え
- 公開すると保護者の画面にテストが表示されます

---

## 4. 生徒管理

**場所：** 管理画面 → 「生徒管理」タブ

### 生徒を追加する
1. 「生徒を追加」をクリック
2. 名前（必須）、学年（必須）、クラス（任意）を入力
3. 「追加」をクリック
4. **アクセスコード**が自動生成されます（英数字6桁）

### アクセスコードの配布
- 生徒一覧に表示されるアクセスコードを保護者にお渡しください
- 保護者はこのコードでログインします

---

## 運用の流れ（まとめ）

```
1. 単元を登録する
2. 問題をDBに登録する（回を重ねるごとに蓄積されていきます）
3. テスト回を作成し、問題を割り当てる
4. テストを「公開」にする
5. 保護者にアクセスコードとURLを伝える
6. 保護者が○×を入力 → 自動で採点・弱点分析・復習教材が生成される
```

## よくある質問

**Q. 問題文は保護者に見えますか？**
A. いいえ。保護者の画面には問番号と配点のみが表示されます。保護者は紙テストを見ながら○×を入力します。

**Q. 配点合計が15点にならない場合は？**
A. テスト作成画面で合計が15点でないと保存できません。各問題の配点を調整してください。

**Q. 同じ問題を複数のテストで使えますか？**
A. はい。問題マスタDBに登録された問題は何度でも各テストに割り当てられます。
